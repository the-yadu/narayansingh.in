# Narayan AI — AI-Native Personal Website

A cinematic, AI-first personal website built with Next.js 15, TypeScript, Tailwind CSS, shadcn-style components, Framer Motion, Zustand, and the Vercel AI SDK with OpenRouter.

## Tech
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Zustand
- Vercel AI SDK + OpenRouter

## Environment Variables
Create `.env.local`:

```bash
OPENROUTER_API_KEY=
OPENROUTER_MODEL=
```

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```
